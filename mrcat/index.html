    🌅 Morning Briefing

    Live disaster scan → AI-generated executive report

    📖 How to use this tool

    Every morning, a CAT Analyst starts by scanning for events that happened overnight. This tool pulls live data from USGS (earthquakes), NWS (weather alerts), and FEMA (disaster declarations), then cross-checks them against your insured portfolio. The AI then writes you a professional executive briefing — the same report your manager expects on their desk by 9am.

    • Choose Days back to control how far back to scan (7 days is standard morning routine).
    • Choose Radius in miles — how close a disaster must be to a policyholder to count as an exposure hit.
    • Click Run Morning Scan. Takes about 10–15 seconds.
    • The report is formatted for copy-paste into an email or PowerPoint.

    📈 PML Report & EP Curve

    Probable Maximum Loss · Average Annual Loss · Exceedance Probability

    📖 How to use this tool

    The PML Report and EP Curve are the two most important technical outputs a CAT Analyst produces. You present these to senior management, actuaries, and reinsurance teams.

    • PML (Probable Maximum Loss) — the largest loss expected to occur in a 1-in-100 or 1-in-250 year event. If a "1-in-100 year hurricane" hits, how much does the company lose?
    • AAL (Average Annual Loss) — the expected loss per year on average, across thousands of simulated years. This is the basis for pricing.
    • EP Curve — a graph showing loss amounts at each return period. Used in reinsurance negotiations and regulatory filings (AM Best, S&P).
    • ⚠️ This is a demonstration model. Real PML uses RMS (Moody's) or AIR (Verisk) platforms.

    Geographic concentration of insured exposure

    📖 How to use this tool

    A critical mid-day task is identifying risk concentration — too many high-value policies in one geographic area. If a single hurricane wipes out a Florida coastal zone and you have 40 policies there, that's catastrophic. This tool shows those hotspots.

    • The map shows each policyholder as a circle sized by insured value.
    • The peril breakdown bar chart shows where your total exposure is concentrated by hazard type.
    • In a real workflow, you'd use ArcGIS or QGIS for this — this tool replicates the core insight.
    • Red circles = highest value policies. Overlapping circles = concentration risk.

    🔍 Data Quality Checker

    Validate & score your exposure data before running models

    📖 How to use this tool

    Data cleaning is 30–50% of a CAT Analyst's actual time. Before running any catastrophe model, the input data (property locations, construction types, insured values) must be clean and complete. Garbage in = garbage out. This tool audits the portfolio and flags issues that would skew your model results.

    • Missing geocodes — properties without lat/lon can't be modeled spatially.
    • Missing construction data — construction type affects vulnerability by 20–40%.
    • Age flags — pre-1980 buildings often lack modern building codes, increasing loss potential.
    • Value outliers — unusually high or low insured values may indicate data entry errors.
    • Fix flagged issues before feeding data into RMS or AIR models.

    AI-powered WRITE / DECLINE recommendation for new submissions

    📖 How to use this tool

    Every afternoon, underwriters bring new property submissions to the CAT Analyst: "Should we insure this hotel in Miami? What's our exposure?". The analyst evaluates the risk against the existing portfolio and says yes, no, or "yes, but raise the premium." This tool automates that analysis.

    🚨 Claims Triage

    Prioritized list of policyholders most likely to file claims today

    📖 How to use this tool

    After a major event, the claims team needs to know who to call first. A CAT Analyst provides a prioritized triage list — ranked by claim likelihood, urgency, and estimated loss — so adjusters can proactively reach out to policyholders before the calls flood in.

    • Immediate — likely filing a claim within 24 hours. Assign a dedicated adjuster now.
    • Same-day — affected by the event, may file soon. Monitor closely.
    • Monitor — marginal exposure, watch but don't mobilize resources yet.
    • The estimated claim range is a rough guide, not a formal reserve. Use vendor models for actual reserves.
